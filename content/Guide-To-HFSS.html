HFSS Terms

Summed input


  1. Project manager contains a design tree which lists the structure of the project. It shows the name of the design, design setup, automation (optimization) and design results.
  2. Message manager tells about the warnings and errors before beginning of the simulation and during the simulation.
  3. Property window tells about the model parameters and attributes. It allows user to change the model parameters.
  4. 3D modeller window displays the model the model tree for active design.
  5. Toolbar contains the shortcut for the tools that can we use directly. We can use these tools from the menu bar. We can customize toolbar according to frequently used tools.
  6. Status bar display the postition of the cursor in the 3D modeller window. It contains coordinate entry field to set coordinates.

Steps involved in patch antenna designing

  1. Select File --> New
               New project will be displayed in project manager

    Summed input


  2. Insert HFSS design from toolbar
               New Modeller Window will be openned

    Summed input


  3. Select the solution type:
    1. Click HFSS --> solution type
    2. Solution type window will display
    3. Select one and press Ok

    Summed input


    1. Draw a Box for substrate in modeller window
    2. Assign material and proper dimensions in object property dialog box

    Summed input


    Each object has two properties :

    1. Attributes
    2. Commands

    By clicking the object in modal tree, attribute dialog box will appear in property window

    Summed input


    Object attribute has following properties

    • Name - user defined name. Default names start with the primitive type followed by an increasing number: Box1, Box2, etc.
    • Material - user defined material property. The default property is vaccum. This can be changed by using the material toolbar.
    • Solve inside - By default HFSS only solves for fields inside dielectrics. To force the HFSS to solve inside conductors, check solve inside.
    • Model Object - Controls if the object is included in the solve.
    • Display Wireframe - Forces the object to always be displayed as wireframe.
    • Colour - Set object colour.
    • Transparency - Set the transparency of an object. 0-Solid, 1- Wireframe.

    Designing parameters of object can be filled by using command window. It is opened by expanding object in modal tree and clicking create.

    Summed input


    We can use variables instead of fixed dimensions.

  5. Draw a rectangle below the substrate and name it as ground.

    Summed input


  6. Draw another rectangle above the substrate and name it as patch.

    Summed input


  7. Draw another rectangle for micro strip transmisssion line and unite it with patch by using boolean algebra.

    Summed input


  8. Draw another rectangle in YZ plane and name it as source.

    Summed input


  9. Draw another Box and name it as Air for the radiation from the structure.

    Summed input


  10. Now further assign the boundary conditions to the different objects. Boundary conditions enable the control of the characteristics of the different objects. It is very important to have the information about the boundary conditions.

    As we know in antenna radiating element and ground should be perfect conductor. Therefore, assign them perfect electrical boundaries.

    Assign boundary to patch:

    Select Patch --> Right click on Patch --> select Boundary --> Per_E

    Summed input


  11. Assign Boundary to Ground by selecting ground plane in same way above.

    Summed input


  12. Assign radiation boundary to air box. Radiation boundaries act as absorbing boundaries and waves can radiate out of the structure and toward the radiation boundary. The system absorbs the wave at the radiation boundary, essentially ballooning the boundary infinitely far away from the structure into space.

    Note: The S-parameter stabilizes at lambda/4, from the radiating element. We recommended that you size the air volume lambda/4 at the lowest frequency. This example has a lower sweep point 2 GHz and a lambda/4 of 4 cm.

    Select Faces of Air Box --> Right Click --> Boundary --> Radiation

    Summed input


  13. Assign excitation

    Select Source --> Right click on Source --> Assign Excitation --> Lumped Port

    A dialog box will appear and use ground as reference then click ok.

    Summed input


  14. Analysis Setup:

    Right click on analysis in Project Manager --> add Solution Setup

    A new window will appear that contains following parameters:

    Solution Frequency solution frequency is desired frequency at which we want results. If the solution frequency is not in the band then results will not be accurate.

    Maximum no. of passes: This number controls the maximum number of passes the adaptive mesh routine will perform as it attempts to satisfy the convergence criteria and to obtain desire results.

    Maximum delta S: This number defines the convergence criteria for the adaptive meshing process. The Delta S is defined as the maximum change in the magnitude of the S-parameters between two consecutive passes.

    Add frequency sweep: right click Solution Setup --> add Frequency Sweep

    A new window will appear that contains following parameters:

    Sweep: HFSS provides following methods to calculate response:
    • Discrete
    • Fast
    • Interpolating

    Each method has own pros and conss.


    Start frequency: Here, we have to define staring frequency for which we want results.

    Stop frerquency: Here, we will define stop frequency as shown in figure.

    Step size: Step size defines the difference between succeccive steps in calculating response.

  15. Check validation of the model by clicking validate button in the toolbar.


    A new window will appear after clicking validate button and tell status of the model. If there is any cross check then it means there is any error which will be shown in message box and after debugging then validate again to recheck.

  16. Run simulation by clicking Analyse all button in the toolbar.


    Clicking analyse all, simulation will start and progressing will be shown in the progress window.

  17. Analyse Result Right click on Results in Project Manager and we can create various parameters such as
    • S parameter
    • Gain
    • Directivity
    • Radiation pattern
    • VSWR and so on

    There are also many other features which makes this software suitable choice for engineers and technicians to design new RF structures such as optimization and parametric sweep.
